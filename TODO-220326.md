# 📝 To-Do List : Optimisation Portfolio Callisto Arts

## 🚀 Performance & Technique

### Optimisation des images

- Convertir les visuels en format `.webp` ou `.avif`.
- Mettre en place le Lazy Loading (`loading="lazy"`).
- Créer des miniatures pour la grille de l'accueil.

### Gestion des particules

- Réduire le nombre de points et de connexions.
- Désactiver les particules sur mobile.
- Respecter `prefers-reduced-motion`.

### Modernisation

- Migrer le projet vers Vite.js (ou Next.js si besoin d'une DB).
- Nettoyer le CSS (supprimer les redondances, utiliser des variables).

---

## 📱 Ergonomie & Responsive

### Adaptation Mobile

- Corriger les erreurs de Flexbox (débordements).
- Rendre le site 100% utilisable sur smartphone (navigation tactile).
- Tester l'affichage sur différents navigateurs.

---

## ⚖️ Accessibilité & Légalité (WCAG / RGPD)

### Accessibilité (WCAG)

- Vérifier les contrastes de couleurs (ratio min 4.5:1).
- Ajouter des balises `alt` descriptives sur toutes les images.
- Rendre la navigation possible au clavier (Tab).

### Conformité Légale

- Ajouter une alerte/bannière de cookies (obligatoire pour l'UE).
- Créer une page de Mentions Légales.
- Sécuriser le formulaire de contact.

---

## 🚩 Choses à ÉVITER (Anti-patterns)

- Ne pas utiliser d'Autoplay Audio.
- Ne pas modifier la vitesse de défilement naturelle (Scroll Hijacking).
- Éviter le texte incrusté directement dans les images.
